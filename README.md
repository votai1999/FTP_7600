# GPRS/3G/4G SIM7600-FTP

### FEATURE
 - List files, folders from FTP server.
 - Download files from FTP server (saver to SD card).
 - Play `.mp3` from SD card.

### PIN MODE
|           |            |
| --------- | ---------- |
| **ESP32** | **7600CE** |
| RX (16)   | TX         |
| TX (17)   | RX         |
